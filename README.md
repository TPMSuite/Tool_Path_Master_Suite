# Tool Path Master Suite (TPMSuite)

## Welcome to the downloading page of TPMSuite, we sincerely hope that you will enjoy it! 

### Updates
ver2.0: Arbitrary three-dimensional functions are supported to define the gradient transition of TPMS lattices. 

### Before getting started, please note the following
1. Run TPMSuite.exe and follow the installation steps. The installer will automatically download the required MATLAB Runtime from the web, which may take a few moments.
2. The slicing result will be saved in the format of Common Layer Interface (CLI), a standard toolpath input format supported by many LPBF printers.
3. The "user_guide_videos" folder provides some videos to help you have a good command of the software.
4. The licenses of the software are in the "licenses" folder.
