# Tool Path Master Suite (TPMSuite)

### Welcome to the downloading page of TPMSuite, we sincerely hope that you will enjoy it! 

### *Before getting started, please note the following:
1. Following the steps of "TPMSuite.exe", the software will be installed smoothly. During the process, the installer will download Matlab Runtime from the web, which may take a few moments.
2. The slicing result will be saved as the format of Common Layer Interface (CLI), which serves as a tool path input file for 3D printers supporting this format. 
3. The "user_guide_videos" folder provides some videos to help you have a good command of the software.
4. The licenses of the software are in the "licenses" folder.
